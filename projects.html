6-legged spider robotfavorite

ณ ปัจจุบัน ดิฉันจบการศึกษาจากมหาวิทยาลัยศิลปากรวิทยาเขตพระราชวังสนามจันทร์ คณะวิศวกรรมศาสตร์และเทคโนโลยีอุตสาหกรรม
โดยดิฉันได้มีความสนใจทางด้านเกี่ยวกับการพัฒนาโค้ดในด้านของซอฟต์แวร์ไม่ว่าจะเป็น หุ่นยนต์หรือตัว Web Application
โปรเจคที่ดิฉันได้รับมอบหมายคือการสร้างซอฟต์แวร์และการจำลองโมเดล "หุ่นยนต์แมลง 6 ขา" หรือมีชื่อโครงงานว่า
"การค้นหาท่าเดินที่เหมาะสมของหุ่นยนต์แมลงหกขาด้วยการจำลองการทำงานเชิงกายภาพ" หรือ "Optimizing The Gait Of A Hexapod Robot Using Physical Simulation"
เพื่อศึกษาเกี่ยวกับการหาท่าเดินที่ดีที่สุดของหุ่นยนต์แมลง 6 ขา โดยจะใช้การหาค่าที่เหมาะที่สุดหรือ PSO ในการ Optimize สำหรับปรับค่าพารามิเตอร์ให้ได้เป็นท่าเดินออกมา
และจำลองบนโปรแกรม Pybullet ซึ่งจะมีเกณฑ์วัดสองเงื่อนไข คือ ขาของตัวหุ่นไม่ชนกันและความเร็วเฉลี่ยที่มีการเคลื่อนที่ได้ไวที่สุด

                สำหรับ URDF (Unified Robot Description Format) คือ รูปแบบไฟล์ XML ที่ใช้สำหรับการจำลองโมเดลหุ่นยนต์ขึ้นมาเพื่ออธิบายลักษณะกายภาพของตัวโมเดลไม่ว่าจะเป็น สี พื้นผิว ตำแหน่ง โดยจะมีการเชื่อมต่อส่วนประกอบต่างๆโดยใช้ Link และ Joint

                สำหรับ Particle Swarm Optimization (PSO) เป็นอัลกอริทึมการหาค่าเหมาะสมที่สุดที่ได้รับแรงบันดาลใจมาจากพฤติกรรมทางธรรมชาติของสัตว์ที่อยู่อาศัยเป็นฝูง
เช่น ฝูงนกหรือฝูงปลา ที่มีการเคลื่อนที่ร่วมกันอย่างเป็นระบบเพื่อค้นหาแหล่งอาหารหรือเป้าหมาย
                PSO ใช้แนวคิดของ อนุภาค (Particle) ซึ่งเป็นตัวแทนของนกหนึ่งตัว โดยแต่ละอนุภาคจะมี ตำแหน่ง (Position) และความเร็ว (Velocity) ในการเคลื่อนที่ไปในพื้นที่ของคำตอบหรือแหล่งอาหาร เพื่อค้นหาคำตอบที่ดีที่สุด อนุภาคแต่ละตัวจะปรับตำแหน่งของตัวเองในแต่ละรอบตามประสบการณ์ของตัวเอง เรียกว่า ค่าที่ดีที่สุดของอนุภาค (pbest: Particle best) และตามประสบการณ์ของฝูง เรียกว่า ค่าที่ดีที่สุดของฝูง (gbest: Global best)

                โดยดิฉันได้ทำการสร้างแบบจำลองของ หุ่นยนต์แมลง 6 ขาขึ้นมา และเรียนรู้วิธีการขยับขาตามต้นแแบบจริง ซึ่งดิฉันทำโดยการจำลองโมเดลบน Rviz
และการเขียนซอฟต์แวร์ในการสร้างโมเดลด้วยโปรแกรม Visual Studio Code โดยจะรันในส่วนของไฟล์ XML ในนามสกุลของ .urdf และนำมาจำลองบนโปรแกรม Pybullet เพื่อค้นหาท่าเดินที่ดีที่สุดโดยใช้ PSO
